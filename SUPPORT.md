# Support

- Product questions and reproducible bugs:
  `https://github.com/Zacxxx/teach/issues`
- Security vulnerabilities: use the repository's private GitHub security
  advisory flow. Do not attach recordings or credentials to a public issue.
- Installation and platform requirements: see
  [README.md](README.md) and
  [documentation/platform-support.md](documentation/platform-support.md).

Teach is an open-source Build Week project. Support is community-based and no
response-time guarantee is offered.
